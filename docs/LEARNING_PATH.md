## 📝 学习建议

### 每天学习流程
1. **阅读学习目标**（5 分钟）
2. **运行示例代码**（15 分钟）
3. **理解核心原理**（30 分钟）
4. **动手实践**（30 分钟）
5. **记录笔记**（10 分钟）

### 遇到困难怎么办？
1. 先看代码注释
2. 运行 `--test` 模式看详细日志
3. 用 AI 助手（比如 Hermes）解释代码
4. 查看相关文档

### 完成标志
- ✅ 代码能运行
- ✅ 能向别人解释原理
- ✅ 能独立修改/扩展功能
